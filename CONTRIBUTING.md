# Contributing

This is a personal project. Contributions are not actively solicited, but feedback and bug reports are welcome.

## Reporting Issues

If you find a bug or have suggestions, please:
1. Check existing issues first
2. Provide clear steps to reproduce
3. Include your environment details (OS, Node version, etc. if applicable)

## Code Changes

This project is not open to external contributions at this time. If you're interested in the project's direction, please open an issue to discuss before submitting a PR.
